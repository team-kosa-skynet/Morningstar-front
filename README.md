# 1. Project Overview (프로젝트 개요)
- 프로젝트 이름: 개발자의 방주
- 프로젝트 설명: 개발자의 방주는 AI 시대에 맞춰 개발자들의 정보 획득과 취업을 도와주는 커뮤니티 사이트 입니다.
- 사이트 이동하기: www.gaebang.site
