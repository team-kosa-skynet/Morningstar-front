일단 ai chat 페이지에서
모델을 선택하고 제출버튼을 눌렀을 때 api를 연동해줘. 리퀘스트 보내는거만 연동해주고
리스폰스 받아서 상세 페이지 표시해주는거는 나중에 처리할거야.

먼저 전송 버튼을 누르면 세션을 생성해야하고, 200과 세션 아이디를 반환받으면
사용자가 선택한 모델에 맞게 각 각 개별 api를 세션 아이디와 함께 리퀘스트로 보내줘야되.
전부 헤더에 토큰도 같이 포함해서 보내줘야되. 세션 생성 api도.
